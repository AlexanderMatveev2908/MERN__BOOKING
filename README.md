# Project Overview

This project includes various implementations and features that I have built as part of my learning process. While many of the approaches used in this project may not be the most efficient or practical for real-world applications, they have been implemented primarily for exercise purposes.

## Key Points:

- **Learning through practice**: The majority of the implementations were done as exercises to better understand the mechanics of libraries and frameworks.
- **Could have been simplified**:Could have been simplified: In many cases, the solutions could have been made simpler or more efficient, but I chose this approach because I believe that a great way to understand a concept is to apply it first in a more simple or controlled environment.
- **User interactivity rights**:For practical reasons every user can be admin as well as a regular user. But in every case only who created a certain hotel can see the option of modifying it from homepage while regular users can only see the hotel details.
